

| Flag | Nom           | Si flag = 1                          | Si flag = 0                           |
|------|---------------------------|--------------------------------------------------|--------------------------------------------------|
| AC   | Retenue auxiliaire        | Retenue depuis le bit 3                         | Aucune retenue depuis le bit 3                   |
| CY   | Retenue (Carry)           | Retenue depuis le bit le plus significatif      | Pas de retenue                                   |
| EI   | Interrupt Enable          | Interruptions matérielles autorisées            | Interruptions matérielles désactivées            |
| OV   | Dépassement (Overflow)    | Dépassement en arithmétique signée              | Aucun dépassement signé                          |
| PE   | Parité (Parity)           | Résultat avec parité paire                      | Résultat avec parité impaire                     |
| PL   | Signe (Sign)              | Résultat négatif (bit de signe = 1)             | Résultat positif (bit de signe = 0)              |
| UP   | Direction (Direction)     | Décrémentation automatique dans les opérations  | Incrémentation automatique                       |
| ZR   | Zéro (Zero)               | Résultat égal à zéro                             | Résultat différent de zéro                       |
